Osnovna sintaksa

Postoje 2 načina rada s Python interpreterom:
interaktivni način rada
Pokretanjem naredbe „python3“ dobivamo ljusku u kojoj izvodimo naredbe.
Npr. funkcija: print(“python“) koju ako ukucamo u interpreteru ispisuje python na ekran.
pokretanje programa pomoću interpretera
Niz naredbi spremimo u datoteku „.py“ te ju pokrenemo interpreterom.
osnovni tipovi varijabli su:
bool, int, long, float, complex
nepromjenjivi: str, bytes, tuple, frozenset
str i bytes su zamjene C++-ovom std::string i char*
tuple je zamjena C++-ovim poljima
promjenjivi: list, set, dict, bytearray
za svaki od tipova postoji istoimena funkcija (tzv. konstruktor)
koja služi za stvaranje instanci klase i pretvaranje među različitim tipovima
osnovni operatori (=, +, -, *, **, /, //, %)
Python zahtjeva točnu uvlaku koda (četiri razmaka ili tab koji automatski pravilno uvlači)
nedostatak uvlake rezultira greškom: „IndentationError: expected an indented block“
višak uvlake: „IndentationError: unexpected indent“
None – tip podataka ništa; (a nedefiniran) != (a = None)
True i False –logički tip podataka;
operatori and i or (u c++ && i ||)
