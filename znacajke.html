Što je Python? Python je interpreterski, objektno-orijentiran, programski jezik visoke razine i dinamičke semantike, veoma pogodan za brzo razvijanje programa kao što je razvoj prototipova, pisanja skripti i povezivanja već postojećih djelova koda. Python je jednostavan za učiti i pogodan za čitanje što znači da ima manje troškove održavanja programa. Python se može besplatno skinuti, koristiti pri pisanje vlastitog programa te se može modificirati redistribuirati (iako je zaštićen copyrightom dostupan je pod licencom otvorenog koda.)
