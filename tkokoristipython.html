Tko jos koristi python?

Corel: PaintShop Pro je proizvod koji mnogi ljudi koriste kako bi snimili snimke zaslona, uređivali svoje slike, crtali nove ili izvodili mnogo drugih zadataka vezanih za grafiku. Nevjerojatna stvar o ovom proizvodu je da se veoma oslanja na Python skriptiranje. Drugim riječima, za automatizaciju zadataka u PaintShop Pro, potrebno je poznavati python.

D-Link: Koriste Python za stvaranje aplikacije za ažuriranje uređaja, Python omogućuje smanjenje broja ponovnog podizanja sustava na samo jedan. D-Link je odabrao Python umjesto drugih jezika, kao što je Java, jer pruža jednostavniji serijski komunikacijski kod.

HP: Korištenjem Python-a olakšavaju podešavanje tražilice kako bi zadovoljila specifične potrebe zaposlenika. Osim toga, Python se pokazao kao bolje rješenje od drugih jezika, kao što je Java.

Philips: Philips koristi Python za sekvencioni jezik (jezik koji govori o koracima koje svaki robot treba poduzeti). Kod niske razine napisan je u C++ (zbog bolje optimizacije) , što je još jedan od razloga za korištenje Pythona, jer Python dobro funkcionira s C++.

United Space Alliance: Ova tvrtka pruža veliku podršku NASA-i za različite projekte, kao što je space shuttle. Jedan od njegovih projekata je stvaranje Workflow automatskog sistem-a (WAS), aplikacije dizajnirane za upravljanje NASA-inim i drugim projektima. Python je odabran umjesto drugih jezika kao što su Java i C++ jer pruža dinamičko tipkanje, jednostavnu sintaksu i interpretera. Rezultat toga je da se aplikacija brže razvija, a pojedinačno testiranje svakog dijela je lakše.
